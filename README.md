# 2023-3e-Zeryk
Žeryk - Sdílejte chutě, nejen recepty

## Instalace
1. Clone repozitáře

    ```bash
    git clone https://github.com/gyarab/2023-3e-Zeryk.git
    ```

2. Přejděte do vytvořené složky

    ```bash
    cd 2023-3e-Zeryk
    ```

3. Nainstalujte Python dependencies

    ```bash
    pip install -r requirements.txt
    ```
